# Dogfood Sarf Map

This map routes repeated Qamus/VN/RICH hover defects to morphology instruction. It is a learner-facing index, not
a new procedure.

| defect class | morphology concept | examples | durable route |
|---|---|---|---|
| finite verb dictionary prose | finite verb features before English | `يَسْأَلُكَ`, `فَأَهْلَكْنَاهُمْ`, `قَالُوا` | `procedures/verb-form-and-mood-review.md`; `drills/dogfood-sarf-remediation.md` |
| suffix pronoun omitted | suffix role depends on host POS | `جَادَلُوكَ`, `أَنزَلْنَاهُ`, `مَنَاكِبِهَا` | `procedures/suffix-pronoun-state.md`; `../nahw/procedures/pronoun-attachment.md` |
| clitic/host collapse | proclitic/article/host/suffix segmentation | `بِسَلَامٍ`, `وَالشَّجَرُ`, `لِتُضَيِّقُوا` | `procedures/clitic-and-host-morphology.md` |
| weak/hamzated/doubled loss | hidden radical recovery | `قَالَ`, `زَاغَ`, `خَفَّت`, `يَسْأَلُ` | `procedures/root-decision.md`; `references/weak-verbs.md` |
| noun/verb/POS collision | POS gates English vocabulary | `رَسُول`, `ذِكْر`, `مُحَمَّد` | `procedures/masdar-participle.md`; `procedures/proper-noun.md` |
| nominal derivative leakage | maṣdar/participle/ṣifa are nouns/adjectives | `مُعَلِّم`, `مُعَلَّم`, maṣdar rows | `curriculum/zero-to-fluency-sarf.md` stage 6 |
| component-only overclaim | segment detection below whole-token certification | `وَالشَّمْسُ`, `بِٱلْمَعْرُوفِ` | `procedures/clitic-and-host-morphology.md`; Qamus morphosyntax contract |

No-op note: weak-root, hamza, doubled-root, verb-form, and derivative procedures already exist. This pass adds the
dogfood map and drills so tutors can route real hover failures without opening tranche reports.
